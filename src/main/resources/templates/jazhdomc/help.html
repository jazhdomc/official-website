FAQ (Frequently Asked Questions)

What's the server IP?

Can I use Hack Client? Can I use a glitch/bug/exploit?

No. Doing so will get you a warning and maybe a ban. Check the rules for more info.

Why doesn't my bubble elevator work? Why can't I edit signs?

You are probably on a server running Minecraft Java Ed. version 1.12.2, the mainly used one. In that version, bubble elevators and signs have not been added yet.

Is the server (proxy) online?

I try to keep the servers and proxy online as much as possible, however sometimes it may be down. You can check the discord server for information about outages.

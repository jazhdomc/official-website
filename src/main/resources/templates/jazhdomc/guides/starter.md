# Starter Guide

## Server Connection
1. Launch your Minecraft or Eaglercraft Java client.
2. Open the server selection menu.
3. Click "Add Server" and enter the Server Name: JazhdoMC and the Server Address: wss://itsjaz.com (for eaglercraft, for java its "itsjaz.com") and then click done.
4. If the server logo and motd loads, the server is online, then you can click the biggest triangle when hovering over the server logo. That will connect you to the server.

## First Login (Registration)
1. You will see a welcome message in chat. Only you see this message because its broadcasted directly to you.
2. You will have two minutes to register and login. Think of a password and then use the commands: "/register &lt;password&gt; &lt;password&gt;" and "/login &lt;password&gt; &lt;password&gt;" to register and login.

## Connecting to a game / backend / server
1. You now can turn around to see 5 NPCs, each with a name, playercount, and a list of 3 different commands you can use to get to the server.
2. Go to one of them and right click them to join that server.

## Server Specific Guides
### Creative Server
1. Use the command "/plot auto" to claim your creative plot and teleport to it.
2. Then use the command "//wand" to get your worldedit wand which you can use to set a large area of blocks to something.

### 1.12 Vanilla Survival Server
1. Mine a tree, get tools, and play normal survival.

### 1.21 Survival Server
1. Use the command "/rtp" to teleport to a random location outside of spawn.
2. Mine a tree, get tools, and play.
3. You can use the command "/help" to look through the many additions to normal survival.
4. To sell items for money, use the command "/sellgui" and to buy, use "/shop"

### Events Server
1. When a announcement of a event appears in the discord server or in the lobby, you can go here to attend it at the specified time. Use the command "/join" to join the event.

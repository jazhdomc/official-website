# Developer Guide

## Contributing
All projects are found on the [GitHub](https://github.com/jazhdomc). You will have to merge with the main branch after coordinating who will be working on what. This guide was made before there were any developers other than Jazhdo, so there may be another method, for example a developing version that is synced across all dev's devices. Make sure to follow the versioning sematics of v&lt;major&gt;.&lt;minor&gt;.&lt;patch&gt;.
